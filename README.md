## 📌 About

Stock Price Visualizer is a data visualization project that analyzes stock prices of top Indian companies like **TATAMOTORS, RELIANCE, INFOSYS, WIPRO, HDFC**. It features multiple chart types, technical indicators, and statistical analysis — all in a dark-themed interactive dashboard.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📈 **Line Chart** | Clean price trend with MA20 & MA50 overlays |
| 🕯️ **Candlestick Style** | OHLC price visualization |
| 🌊 **Area Chart** | Filled area price trend |
| 📊 **Volume Analysis** | Color-coded volume bars |
| 📉 **Returns Distribution** | Histogram + KDE curve |
| 🔥 **Correlation Heatmap** | Feature correlation with Seaborn |
| 📋 **Data Table** | Recent 10-day OHLCV data |

---

## 🛠️ Tech Stack

- **Python** — Core language
- **Streamlit** — Web dashboard
- **Matplotlib** — Charts & visualizations
- **Seaborn** — Heatmap & distribution plots
- **Pandas** — Data manipulation
- **NumPy** — Numerical computations

---

## 🚀 How to Run

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```
